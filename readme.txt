Faça notas ou avisos de modo simples e pratico em seus posts e paginas no WordPress.

== Description ==
Este plugin permite que você crie notas ou avisos agradáveis em seus posts e páginas no WordPress, de modo simples, rápido e pratico usando shortcodes.

Há cinco shortcodes que fazem a mágica acontecer:
**[nota][/nota]**, **[importante][/importante]**, **[dica][/dica]**, **[aviso][/aviso]** e **[ajuda][/ajuda]**.

O WP Notas foi testado nos principais navegadores, como Chrome, Firefox e Opera.

== Installation ==
1. Extraia o conteúdo do arquivo.
2. Envie a pasta `wp-notas` para o diretório `wp-content/plugins`
3. Ative o plugin através do menu 'Plugins' no WordPress
4. Agora é só adicionar as shortcodes.

*Exemplos:*
**[nota]**Olá, mundo!**[/nota]**
**[importante]**É importante manter seus plugins atualizados**[/importante]**
**[dica]**Use filtro solar**[/dica]**
**[aviso]**Não esqueça seus documentos de identificação em casa**[/aviso]**
**[ajuda]**Precisa de ajuda? Fale comigo pelo Twitter, [@tuchelele](https://twitter.com/tuchelele)**[/ajuda]**
